macOS

Mac? Build it yourself

There is no Mac download button here on purpose.

Official excuse from the developer: I am too lazy and do not have enough space on my Mac to install the build tools and ship a .dmg. If you want Desktop Cat on macOS, clone the repo and build it locally — instructions are in BUILD-MAC.md.

  1. Clone the repo (see link below).
  2. On a Mac: install Node, Rust, Xcode CLI tools.
  3. Run npm install, npm run sprites, npm run package:mac.
  4. Open the .dmg from the release/ folder.

FAQ

Where is Desktop Cat after I install?

The cat walks along the bottom of your screen on top of your wallpaper. It is not a normal window in the taskbar. Look for the tray icon in the notification area by the clock (bottom-right on Windows). You may need to click the ^ arrow to show hidden icons.

How do I open settings or quit?

Right-click the tray icon. From there you can change the number of cats, open Cat settings…, reset cats to the bottom, toggle Start at login, or Quit Desktop Cat to close the app completely.

How do I get more than one cat?

Right-click the tray icon → Number of cats → pick 1–8. The change applies right away. You can also use Cat settings… for a slider and − / + buttons, then Apply. Do not launch the desktop shortcut several times — that is one app with one tray icon; use the menu for more cats.

How do I play with the cat?

Click directly on the cat (its pixels, not empty space around it) to pet it or wake it from a nap. Drag a cat to move it; when you let go, it falls back down to the bottom of the screen.

I’m on Mac — why is there no download button?

There is no pre-built Mac installer on this page. Mac users can clone the public repo on GitHub and build a .dmg on a Mac — see BUILD-MAC.md in the repo. After building, use the menu bar icon (top of the screen) the same way you would use the Windows tray menu.
