macOS

Mac? Build it yourself

There is no Mac download button here on purpose.

Official excuse from the developer: I am too lazy and do not have enough space on my Mac to install the build tools and ship a .dmg. If you want Desktop Cat on macOS, clone the repo and build it locally — instructions are in BUILD-MAC.md.

  1. Clone the repo (see link below).
  2. On a Mac: install Node, Rust, Xcode CLI tools.
  3. Run npm install, npm run sprites, npm run package:mac.
  4. Open the .dmg from the release/ folder.

FAQ

Can people download my GitHub repo if it is private?

No. A private repo is only visible to you and people you explicitly invite as collaborators. Random visitors cannot clone it or browse the code. For Mac users who need to clone and build, you should either make the repo public or add each person as a collaborator (impractical for a pet app).

Do I need to make the repo public?

For this website’s Windows button: no — the installer is hosted on Vercel, not GitHub. For strangers who want to build on Mac from Git: yes, unless you only share with friends you add on GitHub.

Why is there no Mac download button?

A Mac .dmg must be built on a Mac with Xcode tools. The Windows .exe is built on Windows and bundled with this site. See BUILD-MAC.md in the repo for the full Mac build guide.
